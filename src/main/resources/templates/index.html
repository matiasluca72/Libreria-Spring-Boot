Acerca de esta app

Este es un proyecto hecho como ejercitación en el lenguaje Java 8 utilizando el framework Spring para diseñar una aplicación con un modelo de diseño MVC. En ella, podrás crear, leer, modificar, y dar de baja datos escenciales de Libros junto a sus Autores y sus Editoriales.
Esta ejercitación forma parte del curso de FullStack Developer de EGG.
